Jarvis

A personal, locally-hosted AI assistant for a single user.

What this is

Jarvis is a private AI assistant running on a Linux server in the operator's home. It is built and operated by a single individual (Ryan Flores) for their own use. It does not offer accounts, does not serve other users, and is not a commercial product.

How Plaid is used

Jarvis uses Plaid to read transaction history and account balances from the operator's own bank, credit union, and credit card accounts. The operator links accounts only to their own personal finances. Read-only access to the transactions product is the sole scope; Jarvis does not request payment initiation, transfer, or any money-movement products.

Data storage

Bank access tokens are encrypted at rest using Fernet (AES-128-CBC with HMAC authentication) on the local Linux server. The encryption key is derived via PBKDF2-SHA256 (600,000 iterations) from a passphrase that exists only on that machine. Transaction and account data live in a local SQLite database on the same server. No financial data is shared with third parties, sent to external analytics services, or transmitted off the operator's network for any purpose other than direct Plaid API calls.

Disconnecting accounts

The operator can revoke Plaid access at any time via their bank's website or via Plaid Portal. Locally, the operator can delete the encrypted access-token entry from their server's vault, which makes any further API calls impossible.
