# Jarvis

Public app-profile page for [Jarvis](https://github.com/ryanjflores6/personal-os), a personal locally-hosted AI assistant.

This repo hosts a single static page describing what Jarvis does, how it uses Plaid, and how data is stored. The page is required by Plaid's Compliance Center for apps that use OAuth-enabled financial institutions.

Page: https://ryanjflores6.github.io/jarvis-app/

For the actual project, see [personal-os](https://github.com/ryanjflores6/personal-os) (private).
